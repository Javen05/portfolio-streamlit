# 🚀 portfolio-streamlit

> A simple yet practical Streamlit e-portfolio template for people who just want a clean portfolio — without overengineering, maintenance, or frontend headaches.

🌐 **Live Demo**: https://javenlai.streamlit.app/

---

## 🧩 Features

* 🧱 Minimal, structured layout
* ⚡ Fast setup (clone → run → done)
* 📝 Easy content editing (no deep coding needed)
* 📂 Organized project sections
* 🎨 Clean, modern UI out of the box
* ☁️ Ready for Streamlit Cloud deployment

---

## 🛠 Tech Stack

* **Streamlit** – UI & app framework
* **Python** – backend logic
* **JAMstack-inspired design** – images and documents are taken from public links/api

---

## 🚀 Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/yourusername/portfolio-streamlit.git
cd portfolio-streamlit
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Replace data

### 4. Run the app

```bash
streamlit run app.py
```

---

## ✏️ Customization

You only need to edit a few things:

* `content/` → your projects, info, links
* `assets/` → images, icons
* `app.py` → layout tweaks (optional)

That’s it.

No need to touch complicated configs.

---

## 📦 Deployment

Easiest way:

1. Push to GitHub
2. Go to Streamlit Cloud
3. Deploy in 1 click

Streamlit supports direct deployment from templates and repos, making it quick to get your app live ([Streamlit][1])
